BEATDOWN (UA2) vs DANGLE (UA1) - SUR BASSKICK
═══════════════════════════════════════════════════════════════════════════════

DIFFÉRENCE ATTENDUE: Beatdown = plus lourd, Master 100 (vs 90 Dangle)

SI SONNE TROP FORT / SAUT VOLUME PERCEPTIBLE:
  → Master: 100 → 95-98 (test réduction légère)
  → Note: "Beatdown Master → 95" pour suite

SI SOUS PAS ASSEZ LOURD / SON TROP SEC:
  → Octave Oct Level: 90 (vérifier, déjà haut) → test 95
  → SI toujours sec: Drive 50 ne change pas, check Compressor Enh Level 30 → 35

SI MANQUE D'AGRESSIVITÉ / PAS ASSEZ MORDANT:
  → Distortion Fx Level: 20 → 22-25 (plus de présence saturation)
  → Drive: 50 reste correct

SI SON BOUEUX / CONFLIT MID-BASS:
  → 4BAND EQ Low-mid: -4dB → -5-6dB (creuse plus les problèmes mids-bas)

PARAMÈTRE CLÉS À NE PAS TOUCHER:
  ✓ Drive: 50 (agressivité Beatdown)
  ✓ Compressor Sustain: 40 (punch court)

DARKGLASS (UA3) vs DANGLE (UA1) - SUR BASSKICK
═══════════════════════════════════════════════════════════════════════════════

DIFFÉRENCE ATTENDUE: Darkglass = technique/transparent, Master 80 (vs 90 Dangle)

SI SONNE PLUS FAIBLE QUE DANGLE:
  → Master: 80 (vérifier, déjà ajusté) → test augmenter 82-85
  → Note: "Darkglass Master → 85" si compensation nécessaire

SI PERTE CLARTÉ / SON ÉTOUFFÉ vs DANGLE:
  → Preamp Middle: -5dB → -8dB (coupe plus mids-haut, laisse aigus passer)
  → High EQ: +8dB (vérifier déjà à +8dB, correct)

SI MANQUE DE PRÉSENCE / COUPURE INCISIVE:
  → Preamp Volume: 90 → 95 (boost léger coloration Ampeg)
  → Preamp Dir Level: 50 → 55 (mélange plus de préamp)

SI TROP GRAIN / TROP AGRESSIF:
  → Preamp Gain: HIGH → vérifier actuellement
  → Si vraiment trop: réduire légèrement Drive 18 → 16

⚠️  DANGER = CONFUSION GUITARE/BASSE:
  Darkglass peut facilement être confondu avec la guitare en live par excès mid-hauts.
  STRATÉGIE ACTUELLE DE SÉPARATION:
    • Hi-mid EQ: -2dB (COUPE zone guitare médiums-hauts)
    • Middle (Preamp): -5dB (coupe médiums globalement)
    • Low EQ: +6dB (basses nettes, guitare faible ici)
    • High EQ: +8dB (aigus présents, mais pas zone guitare)
    • DIR Level (Preamp): 50 (mix équilibré, préamp ne domine pas)
  
  SI CLASH GUITARE PERÇU EN LIVE:
    Réduire agressivement les mid-hauts:
      1. Hi-mid EQ: -2dB → -3dB (creuse plus la zone guitare)
      2. Middle (Preamp): -5dB → -8dB (coupe médiums très sèche)
      3. DIR Level (Preamp): 50 → 40 (moins de préamp color, plus signal direct)
    Test après chaque ajustement = séparation guitare restaurée?

PARAMÈTRES CLÉS À NE PAS TOUCHER:
  ✓ Octave: OFF (clarté pure)
  ✓ Preamp Type: Ampeg (couleur fixture)

⚠️  SOLUTION LARSEN FEEDBACK (testé en live):
  Darkglass peut amorcer un Larsen quand tu te tournes vers l'ampli.
  SOLUTION APPLIQUÉE:
    • DIR Level (Preamp): 45 (réduit 50→45, moins d'amplification préamp)
    • Fx Level (Distortion): 40 (réduit 45→40, moins de saturation feedback)
    • MIC Setting: 8 cm (éloigné 5→8cm, capture moins la boucle feedback)
    • MIC Level: 65 (réduit 75→65, moins d'amplification capteur)
  
  SI LARSEN PERSISTE APRÈS TEST:
    1. MIC Setting: 8 cm → 10 cm (éloigne encore plus)
    2. MIC Level: 65 → 55 (réduit davantage)
    3. Preamp Volume: 90 → 85 (boost coloration moins agressif)
  
  SI LARSEN DISPARU = Configuration stable ! ✅

BREACH (UA4) vs DANGLE (UA1) - SUR BASSKICK
═══════════════════════════════════════════════════════════════════════════════

DIFFÉRENCE ATTENDUE: Breach = hardcore mélodique, Hi-mid +5dB, Master 88

SI SONNE LÉGÈREMENT MOINS FORT:
  → Master: 88 → 90-92 (test compensation légère)
  → Note: "Breach Master → 90" si nécessaire

SI MANQUE DE COUPURE GUITARE / PRÉSENCE INSUFFISANTE:
  → Hi-mid EQ: +5dB (paramètre CLÉ) → [+6-7dB] test priorité absolue
    ⭐ C'EST LA SIGNATURE = en-dessous de +4dB, c'est plus Breach
  → High EQ: +6dB → +7dB (complément aigus)
  → Note: "Breach Hi-mid → +6dB" si boost nécessaire

SI TROP AGRESSIF / PIERCING / MICROPHONIE GUITARE:
  → Hi-mid EQ: +5dB → test réduire à +4dB (minimal)
  → Compressor Attack: 28 → 32 (moins de transitive frappée)

SI PUNCH MANQUANT / NOTES PAS ASSEZ DÉFINIES:
  → Drive: 15 reste correct
  → Compressor Attack: 28 → 22 (frappe plus vif)

PARAMÈTRE IMMUABLE:
  ✓ Hi-mid EQ: MINIMUM +4dB (signature du preset = coupure guitare)

PUNCHYCORE (UA5) vs DANGLE (UA1) - SUR BASSKICK

DIFFÉRENCE ATTENDUE: Punchycore = percussif midrange, Lo-mid +6dB, Master 92

SI SONNE PLUS AGRESSIF QUE DANGLE / PUNCH TROP:
  → Master: 92 → 88-90 (baisse légère pour cohérence)
  → Note: "Punchycore Master → 88" si réduction nécessaire

SI MANQUE DE PUNCH / MIDRANGE PLAT:
  → Lo-mid EQ: +6dB (paramètre CLÉ) → [+7-8dB] test priorité absolue
    ⭐ C'EST LA SIGNATURE = en-dessous de +5dB, perte signature
  → 4BAND EQ Level: +8dB → test +9dB (boost punch global)
  → Note: "Punchycore Lo-mid → +7dB" si boost nécessaire

  → Lo-mid EQ: +6dB → test réduire +4-5dB (minimal)
  → 4BAND EQ Level: +8dB → +7dB

SI PERTE CLARTÉ / SON TROP SOMBRE:
  → High EQ: +4dB → +5-6dB (brille plus)
  → Treble (Distortion): +8 → +9-10 (croustillant)

PARAMÈTRE IMMUABLE:
  ✓ Lo-mid EQ: MINIMUM +5dB (signature punch midrange)
